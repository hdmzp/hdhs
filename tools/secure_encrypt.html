🔐 보안 데이터 암호화 도구

엑셀에서 표(머리글 포함)를 복사해 아래에 붙여넣고, 보안탭 비밀번호를 입력한 뒤 암호화하세요.
모든 처리는 이 브라우저 안에서만 이루어지며 데이터·비밀번호는 어디로도 전송되지 않습니다.

1. 표 데이터 붙여넣기 (첫 줄 = 열 제목)
셀 사이가 탭으로 구분된 형식(엑셀 복사-붙여넣기 기본)과 쉼표(CSV) 모두 인식합니다.
2. 기준일과 비밀번호
※ 사이트 로그인에 쓰는 비밀번호와 반드시 같아야 합니다. 등록한 비밀번호 중 어느 것으로도 로그인할 수 있습니다. 비밀번호 2를 비우면 1개만 등록됩니다.
다운로드한 파일 적용 방법
1) GitHub 저장소의 data/secure 폴더로 이동
2) Add file > Upload files로 방금 받은 secure.enc를 업로드 (같은 이름이라 자동 덮어쓰기)
3) Commit 하면 1~2분 뒤 사이트에 반영됩니다.

비밀번호를 바꾸는 경우: 이 도구에서 새 비밀번호로 다시 암호화해 파일을 교체해야 합니다.
